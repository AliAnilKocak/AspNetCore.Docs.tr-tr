# <a name="migrate-from-aspnet-core-31-to-50"></a>ASP.NET Çekirdek 3.1'den 5.0'a geçirin

Yazar: [Scott Addie](https://github.com/scottaddie)

Bu makalede, varolan bir ASP.NET Core 3.1 projesinin Core 5.0 ASP.NET nasıl güncelleştirilen açıklanmaktadır.

> [!IMPORTANT]
> ASP.NET Core 5.0 şu anda önizleme de.

## <a name="update-net-core-sdk-version-in-globaljson"></a>global.json'da .NET Core SDK sürümünü güncelleyin

Belirli bir .NET Core SDK sürümünü hedeflemek için [global.json](/dotnet/core/tools/global-json) dosyasına güveniyorsanız, `version` özelliği yüklü olan .NET 5.0 SDK sürümüne güncelleyin. Örneğin:

```diff
{
  "sdk": {
-    "version": "3.1.200"
+    "version": "5.0.100-preview.2.20176.6"
  }
}
```

## <a name="update-the-target-framework"></a>Hedef çerçeveyi güncelleştirme

Proje dosyasında, [Hedef Çerçeve Moniker (TFM)](/dotnet/standard/frameworks) `netcoreapp5.0`güncelleştirme:

```diff
<Project Sdk="Microsoft.NET.Sdk.Web">

  <PropertyGroup>
-    <TargetFramework>netcoreapp3.1</TargetFramework>
+    <TargetFramework>netcoreapp5.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## <a name="update-package-references"></a>Paket referanslarını güncelleştir

Proje dosyasında, her `Microsoft.AspNetCore.*` `Microsoft.Extensions.*` ve paket `Version` referansın özniteliğini 5.0.0 veya sonraki olarak güncelleştirin. Örneğin:

```diff
<ItemGroup>
-    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="3.1.2" />
-    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="3.1.2" />
+    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="5.0.0-preview.2.20167.3" />
+    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="5.0.0-preview.2.20160.3" />
</ItemGroup>
```

## <a name="update-docker-images"></a>Docker görüntülerini güncelleştir

Docker kullanan uygulamalar için, Core 5.0 ASP.NET içeren bir temel görüntü kullanın. Örneğin:

```bash
docker pull mcr.microsoft.com/dotnet/core/aspnet:5.0
```

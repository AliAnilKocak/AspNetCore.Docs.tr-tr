  ```xml
  </ItemGroup>
  ```

  <span data-ttu-id="03989-112">Daha fazla bilgi için [NuGet/giriş sorun #7342](https://github.com/NuGet/Home/issues/7342).</span><span class="sxs-lookup"><span data-stu-id="03989-112">For more information, see [NuGet/Home issue #7342](https://github.com/NuGet/Home/issues/7342).</span></span>

## <a name="jsonnet-support"></a><span data-ttu-id="03989-122">Json.NET desteği</span><span class="sxs-lookup"><span data-stu-id="03989-122">Json.NET support</span></span>

<span data-ttu-id="03989-123">İşin bir parçası olarak [ASP.NET Core paylaşılan çerçeve geliştirmek](https://blogs.msdn.microsoft.com/webdev/2018/10/29/a-first-look-at-changes-coming-in-asp-net-core-3-0/), [Json.NET](https://www.newtonsoft.com/json/help/html/Introduction.htm) ASP.NET Core paylaşılan Framework'ten kaldırıldı.</span><span class="sxs-lookup"><span data-stu-id="03989-123">As part of the work to [improve the ASP.NET Core shared framework](https://blogs.msdn.microsoft.com/webdev/2018/10/29/a-first-look-at-changes-coming-in-asp-net-core-3-0/), [Json.NET](https://www.newtonsoft.com/json/help/html/Introduction.htm) has been removed from the ASP.NET Core shared framework.</span></span>

<span data-ttu-id="03989-124">ASP.NET Core 3.0 projesinde Json.NET kullanmak için:</span><span class="sxs-lookup"><span data-stu-id="03989-124">To use Json.NET in an ASP.NET Core 3.0 project:</span></span>

    ```csharp

    ```

<span data-ttu-id="03989-127">Newtonsoft ayarları ile ayarlanabilir `AddNewtonsoftJson`:</span><span class="sxs-lookup"><span data-stu-id="03989-127">Newtonsoft settings can be set with `AddNewtonsoftJson`:</span></span>

    ```csharp

        .AddNewtonsoftJson(options => 
               options.SerializerSettings.ContractResolver = 
                  new CamelCasePropertyNamesContractResolver());
    ```

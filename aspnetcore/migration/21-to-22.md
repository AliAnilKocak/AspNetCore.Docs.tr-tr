# <a name="migrate-from-aspnet-core-21-to-22"></a><span data-ttu-id="19d1f-103">ASP.NET Core 2.1 için 2.2 geçiş</span><span class="sxs-lookup"><span data-stu-id="19d1f-103">Migrate from ASP.NET Core 2.1 to 2.2</span></span>

<span data-ttu-id="19d1f-105">Bu makalede, ASP.NET Core 2.2 için mevcut bir ASP.NET Core 2.1 projeyi güncelleştirin açıklanmaktadır.</span><span class="sxs-lookup"><span data-stu-id="19d1f-105">This article explains how to update an existing ASP.NET Core 2.1 project to ASP.NET Core 2.2.</span></span>

## <a name="update-target-framework-moniker-tfm"></a><span data-ttu-id="19d1f-106">Hedef Çerçeve adı (TFM) güncelleştirme</span><span class="sxs-lookup"><span data-stu-id="19d1f-106">Update Target Framework Moniker (TFM)</span></span>

<span data-ttu-id="19d1f-107">.NET Core'u hedefleyen projeler kullanması gereken [TFM](/dotnet/standard/frameworks#referring-to-frameworks) büyüktür veya eşittir .NET Core 2.2 sürümü.</span><span class="sxs-lookup"><span data-stu-id="19d1f-107">Projects targeting .NET Core should use the [TFM](/dotnet/standard/frameworks#referring-to-frameworks) of a version greater than or equal to .NET Core 2.2.</span></span> <span data-ttu-id="19d1f-108">Güncelleştirme `<TargetFramework>` düğümün iç metni ile `netcoreapp2.2`:</span><span class="sxs-lookup"><span data-stu-id="19d1f-108">Update the `<TargetFramework>` node's inner text with `netcoreapp2.2`:</span></span>

```xml
<TargetFramework>netcoreapp2.2</TargetFramework>
```

<span data-ttu-id="19d1f-109">.NET Framework'ü hedefleyen projeleri TFM büyüktür veya eşittir .NET Framework 4.6.1 sürümü kullanmaya devam edebilir:</span><span class="sxs-lookup"><span data-stu-id="19d1f-109">Projects targeting .NET Framework may continue to use the TFM of a version greater than or equal to .NET Framework 4.6.1:</span></span>

```xml
<TargetFramework>net461</TargetFramework>
```

## <a name="update-net-core-sdk-version-in-globaljson"></a><span data-ttu-id="19d1f-110">.NET Core SDK'sı sürümünü global.json güncelleştirin</span><span class="sxs-lookup"><span data-stu-id="19d1f-110">Update .NET Core SDK version in global.json</span></span>

<span data-ttu-id="19d1f-111">Çözümünüze bağlı dayanıyorsa bir [global.json](/dotnet/core/tools/global-json) dosya belirli bir .NET Core SDK sürümünü hedeflemek, güncelleştirme, `version` özelliği makinenizde yüklü 2.2 sürüm:</span><span class="sxs-lookup"><span data-stu-id="19d1f-111">If your solution relies upon a [global.json](/dotnet/core/tools/global-json) file to target a specific .NET Core SDK version, update its `version` property to the 2.2 version installed on your machine:</span></span>

```json
{
  "sdk": {
    "version": "2.2.100"
  }
}
```

## <a name="update-package-references"></a><span data-ttu-id="19d1f-112">Güncelleştirme paketi başvuruları</span><span class="sxs-lookup"><span data-stu-id="19d1f-112">Update package references</span></span>

<span data-ttu-id="19d1f-113">.NET Core'u hedefleyen, kaldırmanız `Version` metapackage başvuru özniteliği.</span><span class="sxs-lookup"><span data-stu-id="19d1f-113">If targeting .NET Core, remove the `Version` attribute for the metapackage reference.</span></span> <span data-ttu-id="19d1f-114">Dahilini bir `Version` özniteliği aşağıdaki uyarıyı sonuçlanır:</span><span class="sxs-lookup"><span data-stu-id="19d1f-114">Inclusion of a `Version` attribute results in the following warning:</span></span>

```console
A PackageReference to 'Microsoft.AspNetCore.App' specified a Version of `2.2.0`. Specifying the version of this package is not recommended. For more information, see https://aka.ms/sdkimplicitrefs
```

<span data-ttu-id="19d1f-115">Metapackage başvuru şuna benzer olmalıdır `<PackageReference />` düğüm:</span><span class="sxs-lookup"><span data-stu-id="19d1f-115">The metapackage reference should resemble the following `<PackageReference />` node:</span></span>

```xml
<ItemGroup>
  <PackageReference Include="Microsoft.AspNetCore.App" />
</ItemGroup>
```

<span data-ttu-id="19d1f-116">.NET Framework'ü hedefleyen, her paketi başvurusunun güncelleştirme `Version` 2.2.0 veya sonraki bir sürüme özniteliği.</span><span class="sxs-lookup"><span data-stu-id="19d1f-116">If targeting .NET Framework, update each package reference's `Version` attribute to 2.2.0 or later.</span></span> <span data-ttu-id="19d1f-117">.NET Framework'ü hedefleyen tipik bir ASP.NET Core 2.2 projesinde paket başvuruları şunlardır:</span><span class="sxs-lookup"><span data-stu-id="19d1f-117">Here are the package references in a typical ASP.NET Core 2.2 project targeting .NET Framework:</span></span>

```xml
<ItemGroup>
  <PackageReference Include="Microsoft.AspNetCore" Version="2.2.0" />
  <PackageReference Include="Microsoft.AspNetCore.CookiePolicy" Version="2.2.0" />
  <PackageReference Include="Microsoft.AspNetCore.HttpsPolicy" Version="2.2.0"/>
  <PackageReference Include="Microsoft.AspNetCore.Mvc" Version="2.2.0" />
  <PackageReference Include="Microsoft.AspNetCore.StaticFiles" Version="2.2.0" />
</ItemGroup>
```

## <a name="update-compatibility-version"></a><span data-ttu-id="19d1f-118">Güncelleştirme uyumluluk sürümü</span><span class="sxs-lookup"><span data-stu-id="19d1f-118">Update compatibility version</span></span>

<span data-ttu-id="19d1f-119">Uyumluluk sürümünü güncelleştirin `Startup.ConfigureServices` için `Version_2_2`:</span><span class="sxs-lookup"><span data-stu-id="19d1f-119">Update the compatibility version in `Startup.ConfigureServices` to `Version_2_2`:</span></span>


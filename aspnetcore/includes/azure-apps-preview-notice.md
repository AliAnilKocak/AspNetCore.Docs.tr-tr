> [!IMPORTANT]
> **Azure App Service ASP.NET Core 3,0**
>
> ASP.NET Core 3,0 henüz Azure App Service dağıtılmadı. Yakında Azure App Service ASP.NET Core 3 ' ü sağlamanızı umuyoruz. ASP.NET Core 3,0 kullanan bir uygulamayı barındırmak için:

* Azure App Service dağıtımı için bir önizleme sürümü gibi ASP.NET Core 3,0 ' i değerlendirin.
* [Azure App Service için bkz. dağıtım ASP.NET Core önizleme sürümü](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service).

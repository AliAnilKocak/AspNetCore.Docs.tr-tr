### <a name="bi-directional-streaming-call"></a><span data-ttu-id="4aa40-153">İki yönlü akış çağrısı</span><span class="sxs-lookup"><span data-stu-id="4aa40-153">Bi-directional streaming call</span></span>

<span data-ttu-id="4aa40-154">Çift yönlü bir akış *çağrısı, istemci* bir ileti göndermeden başlatılır.</span><span class="sxs-lookup"><span data-stu-id="4aa40-154">A bi-directional streaming call starts *without* the client sending a message.</span></span> <span data-ttu-id="4aa40-155">İstemci, ile `RequestStream.WriteAsync`ileti göndermek için seçim yapabilir.</span><span class="sxs-lookup"><span data-stu-id="4aa40-155">The client can choose to send messages with `RequestStream.WriteAsync`.</span></span> <span data-ttu-id="4aa40-156">Hizmetten akışa alınan iletilere veya `ResponseStream.MoveNext()` `ResponseStream.ReadAllAsync()`ile erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="4aa40-156">Messages streamed from the service are accessible with `ResponseStream.MoveNext()` or `ResponseStream.ReadAllAsync()`.</span></span> <span data-ttu-id="4aa40-157">İki yönlü akış çağrısı, `ResponseStream` daha fazla ileti olmadığında tamamlanır.</span><span class="sxs-lookup"><span data-stu-id="4aa40-157">The bi-directional streaming call is complete when the `ResponseStream` has no more messages.</span></span>

```csharp
using (var call = client.Echo())
{
    Console.WriteLine("Starting background task to receive messages");
    var readTask = Task.Run(async () =>
    {
        await foreach (var response in call.ResponseStream.ReadAllAsync())
        {
            Console.WriteLine(response.Message);
            // Echo messages sent to the service
        }
    });

    Console.WriteLine("Starting to send messages");
    Console.WriteLine("Type a message to echo then press enter.");
    while (true)
    {
        var result = Console.ReadLine();
        if (string.IsNullOrEmpty(result))
        {
            break;
        }

        await call.RequestStream.WriteAsync(new EchoMessage { Message = result });
    }

    Console.WriteLine("Disconnecting");
    await call.RequestStream.CompleteAsync();
    await readTask;
}
```

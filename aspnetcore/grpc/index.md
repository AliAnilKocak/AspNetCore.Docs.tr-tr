<span data-ttu-id="3b538-126">gRPC hizmet proje şablonu bir başlangıç hizmeti sağlar:</span><span class="sxs-lookup"><span data-stu-id="3b538-126">The gRPC service project template provides a starter service:</span></span>

```csharp
public class GreeterService : Greeter.GreeterBase
{
    private readonly ILogger<GreeterService> _logger;

    public GreeterService(ILogger<GreeterService> logger)
    {
        _logger = logger;
    }

    public override Task<HelloReply> SayHello(HelloRequest request,
        ServerCallContext context)
    {
        _logger.LogInformation("Saying hello to {Name}", request.Name);
        return Task.FromResult(new HelloReply 
        {
            Message = "Hello " + request.Name
        });
    }
}
```

<span data-ttu-id="3b538-127">`GreeterService``GreeterBase` `Greeter` *.proto dosyasındaki hizmetten oluşturulan türden \** devralır.</span><span class="sxs-lookup"><span data-stu-id="3b538-127">`GreeterService` inherits from the `GreeterBase` type, which is generated from the `Greeter` service in the *\*.proto* file.</span></span> <span data-ttu-id="3b538-128">Hizmet *Startup.cs*istemciler için erişilebilir hale getirilir:</span><span class="sxs-lookup"><span data-stu-id="3b538-128">The service is made accessible to clients in *Startup.cs*:</span></span>

```csharp

{
    endpoints.MapGrpcService<GreeterService>();
});
```

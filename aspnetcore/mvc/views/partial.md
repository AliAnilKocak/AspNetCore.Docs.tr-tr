### <a name="partial-view-discovery"></a><span data-ttu-id="16815-135">Kısmi görünüm bulma</span><span class="sxs-lookup"><span data-stu-id="16815-135">Partial View Discovery</span></span>

<span data-ttu-id="16815-136">Kısmi görünüm başvururken çeşitli şekillerde konumuna başvurabilir:</span><span class="sxs-lookup"><span data-stu-id="16815-136">When referencing a partial view, you can refer to its location in several ways:</span></span>

```text

// If none is found, searches the Shared folder
@Html.Partial("ViewName")

// A view with this name must be in the same folder
@Html.Partial("ViewName.cshtml")

// Locate the view based on the application root
// Paths that start with "/" or "~/" refer to the application root

// Locate the view using relative paths
@Html.Partial("../Account/LoginPartial.cshtml")
```

## <a name="partial-view-discovery"></a><span data-ttu-id="0a39e-159">Kısmi görünüm bulma</span><span class="sxs-lookup"><span data-stu-id="0a39e-159">Partial view discovery</span></span>

```cshtml
// Locate the view using a relative path

```

<span data-ttu-id="0a39e-162">Yukarıdaki örnekte, ASP.NET Core 2.1 veya üzeri gerektiren kısmi etiket Yardımcısı, kullanır.</span><span class="sxs-lookup"><span data-stu-id="0a39e-162">The preceding example uses the Partial Tag Helper, which requires ASP.NET Core 2.1 or later.</span></span> <span data-ttu-id="0a39e-163">Aşağıdaki örnek aynı görevi başarmak için zaman uyumsuz HTML Yardımcıları kullanır.</span><span class="sxs-lookup"><span data-stu-id="0a39e-163">The following example uses asynchronous HTML Helpers to accomplish the same task.</span></span>

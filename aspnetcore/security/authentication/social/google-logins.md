# <a name="aspnet-core-2xtabaspnetcore2x"></a>[<span data-ttu-id="3bb2f-147">ASP.NET Core 2.x</span><span class="sxs-lookup"><span data-stu-id="3bb2f-147">ASP.NET Core 2.x</span></span>](#tab/aspnetcore2x/)

```csharp

        .AddEntityFrameworkStores<ApplicationDbContext>()
        .AddDefaultTokenProviders();

});
```

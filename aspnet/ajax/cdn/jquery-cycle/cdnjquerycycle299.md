<a name="jquery-cycle-299"></a>JQuery döngüsü 2.99
====================
Aşağıdaki jQuery döngüsü dosyaları bu CDN üzerinde barındırılan:

- https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js &mdash; JQuery döngüsü eklentisi tam sürümü. Birden çok iki düzine özel etkiler ve tüm seçenekleri içerir.
- https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js &mdash; Yukarıdaki dosyanın küçültülmüş sürümü. Web sitesi performansını artırmak için bu sürümü üretim uygulamaları için kullanın.
- https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js &mdash; JQuery döngüsü eklentisi bu sürümü daha küçüktür ve yalnızca "yavaşça" geçiş etkisi içerir.
- https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js &mdash; Yukarıdaki dosyanın küçültülmüş sürümü. Web sitesi performansını artırmak için bu sürümü üretim uygulamaları için kullanın.
- https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js &mdash; Yalnızca "yavaşça" geçişi olmadan tüm seçenekleri içeren jQuery döngüsü eklentisi bile daha küçük bir sürümü.
- https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js &mdash; Yukarıdaki dosyanın küçültülmüş sürümü. Web sitesi performansını artırmak için bu sürümü üretim uygulamaları için kullanın.

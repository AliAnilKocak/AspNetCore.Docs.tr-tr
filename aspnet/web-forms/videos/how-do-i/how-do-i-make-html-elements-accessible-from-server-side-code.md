<a name="how-do-i-make-html-elements-accessible-from-server-side-code"></a>[Nasıl stop yaparım] Sunucu tarafı kodu HTML öğeleri erişilebilir
====================
tarafından [Chris Pels](https://twitter.com/chrispels)

Bu videoda Chris Pels gibi standart bir ASP.NET sunucu denetimi işleme sayfasındaki kullanılabilmesi için standart HTML öğeleri bir .aspx sayfasında sunucu tarafı kodu erişilebilir yapmak nasıl yapacağınızı gösterir. İlk olarak, bkz: nasıl runat eklenmesi = "server" özniteliğini gibi standart bir HTML denetlemek için bir &lt;div&gt; sunucu tarafı kodu IntelliSense ile başvurulan izin verir. Ardından, bu teknik bir sayfada HTML öğeleri nasıl işlemek, çeşitli örnekler bakın. Son olarak, nasıl Bu teknik birlikte .NET Framework sürüm 3.5 yeni ListView denetiminde bir listesinin yerleşimini denetlemek için kullanılabilir bakın.

<a name="enabling-full-text-search-in-your-text-data"></a>Tam metin araması metin verilerinizde etkinleştirme
====================
tarafından [Microsoft](https://github.com/microsoft)

Tam metin araması, hatta Microsoft Word belgeleri veya diğer dosya biçimleri - metin - büyük bölümünü veritabanınıza kaydedip harika performans beklenirken metinsel bu verileri temel alan karmaşık sorgular gerçekleştirmek sağlar. Bu ders belirli dikkat SQL Server Express içinde kurma nüanslarını için tam metin araması ayarlanacağı gösterilmiştir. Daha sonra tam metin dizin oluşturma, dizin popülasyonu ayarlarını yapılandırma ve CONTAINS koşulu ve birçok, birçok Çeşitlemeler gücünü gösteren sorguları gerçekleştirmek için taşıyın.

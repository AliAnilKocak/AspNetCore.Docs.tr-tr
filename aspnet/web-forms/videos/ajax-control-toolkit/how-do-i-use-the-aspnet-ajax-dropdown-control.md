<a name="how-do-i-use-the-aspnet-ajax-dropdown-control"></a>[Nasıl stop yaparım] ASP.NET AJAX açılır denetimi kullanılsın mı?
====================
tarafından [CAN Stagner](https://github.com/JoeStagner)

ASP.NET AJAX Denetim araç setinden açılır denetim extender ile başka bir Web sayfasında bir kısmi sayfa güncelleştirmesini tetiklemek seçenekler aşağı açılan listesi ile kullanıcı sunabilir. Kullanıcı tarafından seçilen seçeneği kısmi sayfa güncelleştirmesi tamamlanmadan önce seçimi işlemek, sunucu tarafı kodu izin vererek sunucuya geçirilir.

<a name="how-do-i-use-javascript-to-refresh-an-aspnet-ajax-updatepanel"></a>[Nasıl stop yaparım] JavaScript, ASP.NET AJAX UpdatePanel yenilemek için kullanılır?
====================
tarafından [CAN Stagner](https://github.com/JoeStagner)

Bu videoda bir ASP.NET AJAX UpdatePanel denetiminin içeriği yenilemek için JavaScript mantığı kullanarak iki farklı yöntemler öğreneceksiniz. İlk yol UpdatePanel bir düğme türü sunucu denetimi ekleyin ve bir zaman uyumsuz geri gönderme tetikler ve 'geri gönderme UpdatePanel içinde sunucu denetimi tarafından kaynaklanmıştır içeriğini' JavaScript kullanmaktır. İkinci yol UpdatePanel eklenecek herhangi bir düğme türü sunucu denetimi gerektirmez, ancak hala bize UpdatePanel içeriği yenilemek için JavaScript mantığı kullanılacak verir. (İkinci tekniği Dave Atla tarafından önerilen ve kendi Web günlüğü girişinde "kolayca JavaScript kullanarak bir UpdatePanel Yenile" daha fazla bilgi edinebilirsiniz.)
